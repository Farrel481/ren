# What is Ren?
### And what is NOT Ren?

## What is Ren?

Hello, my name is Farrel and this is my personal project called REN. So, this project is an Artificial Intelligence
creation which have some differ with usual AI like some we know called ChatGPT, Claude, or any Chat based AI. Ren will help us on daily jobs DIRECTLY, yes, direct, and that is not even the good part.

REN is a companion, designed to feel present. It helps with daily task, learns and grows alongside it's user.

It Evolves.

## What is Not Ren.

- Not a replacement for any kind of Relationship.
- It's not an autonomous entity that acts without permission.
- Not just another chatBots.
- Will not control it's user, it'll ask for opinion.

I work on this project in intention of making my workdesk more alive.

'Course, it'll help me on my Future too, as a portfolio, as a college project, but my main goal is written.
